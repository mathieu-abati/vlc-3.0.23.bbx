# VLC BuildBox package source

This is the package sources to build VLC 3.0.23 in
[BuildBox](https://buildbox.trusted-objects.com).

It is part of [VLC BuildBox project](https://github.com/mathieu-abati/vlc.bbx).
Please use the project directly.

The following changes have been made to VLC 3.0.23 official archive:
- add `build.sh` script, used by
  [BuildBox custom build mode](https://buildbox.trusted-objects.com/user/package.html#custom),
- add `clean.sh` script, also used by BuildBox.
